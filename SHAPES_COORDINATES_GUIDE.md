# Руководство по созданию координат для фигур

## Формат Excel файла

Создайте файл `shapes_coordinates.xlsx` в папке проекта с координатами для каждой фигуры.

### Структура файла:

1. **Создайте отдельные листы** для каждой фигуры:
   - `star` - для звезды
   - `sphere` - для шара
   - `pattern` - для узора
   - `text` - для слова "АРТ"

2. **В каждом листе должны быть колонки:**
   - `index` - порядковый номер иконки (начиная с 0, 1, 2, 3...)
   - `x` - координата X
   - `y` - координата Y
   - `z` - (опционально) координата Z для эффекта глубины
     - **0** = ближе к зрителю (иконка будет **больше** и на **переднем плане**)
     - **1** = дальше от зрителя (иконка будет **меньше** и на **заднем плане**)
     - Можно использовать любые значения - они автоматически нормализуются от 0 до 1

## Как создать координаты

### Вариант 1: Вручную в Excel
1. Откройте Excel
2. Создайте новый файл `shapes_coordinates.xlsx`
3. Создайте листы для каждой фигуры
4. Заполните координаты

### Вариант 2: Использовать графический редактор
1. Нарисуйте фигуру в любом редакторе (Figma, Illustrator, даже Paint)
2. Определите координаты точек
3. Перенесите в Excel

### Вариант 3: Использовать онлайн инструменты
- Можно использовать онлайн калькуляторы для геометрических фигур
- Или создать фигуру в Processing/p5.js и экспортировать координаты

## Обработка координат

После создания Excel файла запустите:

```bash
python process_shapes.py
```

Это создаст файл `shapes_coordinates.json`, который будет использоваться сайтом.

## Советы

1. **Масштабирование**: Координаты можно задавать в любых единицах, они будут масштабироваться автоматически
2. **Центрирование**: Фигуры автоматически центрируются относительно центра доски
3. **Количество иконок**: Если у вас 248 иконок, создайте координаты для всех или только для части (остальные разместятся случайно)

## Примеры координат

### Звезда (5 вершин):
```
Внешние точки (радиус R):
- (0, -R)
- (R*cos(18°), -R*sin(18°))
- (R*cos(54°), R*sin(54°))
- (-R*cos(54°), R*sin(54°))
- (-R*cos(18°), -R*sin(18°))

Внутренние точки (радиус r):
- Между каждой парой внешних точек
```

### Шар (сферическое расположение):
```
Для каждой иконки:
- angle = (index / total) * 2π
- radius = base_radius * (1 - layer * 0.3)
- x = cos(angle) * radius
- y = sin(angle) * radius
```

Могу помочь создать шаблон Excel файла с примерными координатами!
