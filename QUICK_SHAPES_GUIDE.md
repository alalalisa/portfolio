# Быстрая инструкция: Как передать координаты фигур

## Шаг 1: Создайте Excel файл

Я уже создал для вас шаблон: `shapes_coordinates_template.xlsx`

Откройте этот файл в Excel - там уже есть примеры координат для всех фигур.

## Шаг 2: Отредактируйте координаты

В файле есть 4 листа:
- **star** - для звезды
- **sphere** - для шара  
- **pattern** - для узора
- **text** - для слова "АРТ"

В каждом листе колонки:
- **index** - номер иконки (0, 1, 2, 3...)
- **x** - координата X
- **y** - координата Y
- **z** - (опционально) координата Z для эффекта глубины
  - **0** = ближе к зрителю (иконка будет больше и на переднем плане)
  - **1** = дальше от зрителя (иконка будет меньше и на заднем плане)
  - Можно использовать любые значения - они автоматически нормализуются

### Как определить координаты:

1. **Нарисуйте фигуру** в любом редакторе (Figma, Illustrator, даже Paint)
2. **Определите координаты** точек относительно центра (0, 0)
3. **Введите в Excel** - каждая строка = одна иконка

### Примеры:

**Звезда:**
- Внешние точки: (0, -200, 0), (190, -62, 0), (118, 162, 0), (-118, 162, 0), (-190, -62, 0)
- Внутренние точки: между внешними с z=0.8 (дальше)

**Шар (с эффектом глубины):**
- Центр шара: (0, 0, 0) - ближе, больше
- Средний слой: (150, 0, 0.5), (106, 106, 0.5), (0, 150, 0.5)...
- Внешний слой: (200, 0, 1), (141, 141, 1), (0, 200, 1)... - дальше, меньше

**Пример для шара в Excel:**
| index | x    | y    | z    |
|-------|------|------|------|
| 0     | 0    | 0    | 0    | (центр, ближе/больше)
| 1     | 150  | 0    | 0.3  |
| 2     | 106  | 106  | 0.3  |
| 3     | 0    | 150  | 0.6  |
| 4     | -106 | 106  | 0.6  |
| 5     | -150 | 0    | 0.9  | (дальше/меньше)

**Слово "АРТ":**
- Нарисуйте буквы точками
- Каждая точка = одна иконка

## Шаг 3: Сохраните файл

Сохраните файл как **`shapes_coordinates.xlsx`** (без слова "template")

## Шаг 4: Обработайте файл

Запустите в терминале:
```bash
python process_shapes.py
```

Это создаст файл `shapes_coordinates.json`, который будет использоваться сайтом.

## Шаг 5: Обновите сайт

Обновите страницу в браузере (F5) - фигуры будут использовать ваши координаты!

## Важные моменты:

1. **Не обязательно заполнять все 248 иконок** - можно только часть, остальные разместятся случайно
2. **Координаты масштабируются автоматически** - можно использовать любые числа
3. **Центр координат = центр доски** - фигуры автоматически центрируются
4. **Можно редактировать координаты** в Excel и перезапускать `process_shapes.py`
